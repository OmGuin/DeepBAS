## Code Explanation

`train_lim.py`: Train linear model. Arguments: learning rate, batch size, epochs, output bins, inital beta, save, progress bar
- Example usage: `python train_lim.py --lr=2e-4 --batch_size=32, --num_epochs=100, --output_bins=50, --init_beta=0.1`
- Example usage: `python train_lim.py --no-save --no-prog_bar`
